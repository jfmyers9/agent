# Proposal Coherence

Apply only when a relevant proposal or legacy design is available.

- Compare implemented behavior, interfaces, invariants, and data flow with the
  source decision and acceptance criteria.
- Flag omitted criteria, incompatible drift, and unrecorded scope changes.
- Accept implementation details and additions that preserve the contract.
- Keep architecture/style observations in the core lenses.

Return source section, implementation path, divergence, and required
reconciliation.
