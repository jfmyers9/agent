# Agent Configuration Repository Instructions

@global/AGENTS.md

## Repository Neutrality

- Keep this agent-configuration repository company-neutral and reusable.
- Do not add employer- or client-specific names, domains, internal paths,
  systems, codenames, data, or workflow assumptions to tracked files or commit
  messages.
- Use neutral placeholders in examples and tests. Reference public tools and
  vendors only when required for an integration or dependency.
